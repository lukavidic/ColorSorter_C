◆ outputCompareConfig()

void outputCompareConfig ( void )

Configure PWM output compare and Timer2 for servo control.

Sets up Timer2 prescaler and period for 50Hz, initializes output compare registers, and starts the timer to generate PWM signal on OC1.

Configure PWM output compare and Timer2 for servo control.

Resets Timer2, sets prescaler to 1:8, configures PR2 for 50Hz PWM, initializes OC1R and OC1RS to middle position, sets OC1 mode to edge-aligned PWM, selects Timer2 as clock source, and starts Timer2.
